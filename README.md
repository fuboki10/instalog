# Installation

## Server
- ```cd server```
- ```npm install```
- Create `.env` file to add the environment variables
- open `prisma/schema.prisma` and either set the `DATABASE_URL` inside the .env var to run the postgresql database or use the sqlite part
- ```npm run dev```

## Client
- ```cd server```
- ```npm install```
- ```npm run dev```
